WHAT THIS FIX DOES

1. Removes the text "Swipe to see all images" from:
   - the cart page
   - the bottom of the homepage

2. Moves the cart quantity button lower, into the empty square area you marked.

HOW TO USE IT

1. Upload this file to GitHub:
   simple-scents-cart-homepage-fix.js

2. Open index.html and paste this right before </body>:
   <script src="simple-scents-cart-homepage-fix.js"></script>

3. Open cart.html and paste the same line right before </body>:
   <script src="simple-scents-cart-homepage-fix.js"></script>

4. Commit changes.

If the quantity button needs to go a little more down, open the JS file and change:
translateY(48px)

to:
translateY(60px)
