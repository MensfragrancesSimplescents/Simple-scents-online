How to use this file:

1. Upload buy-now-direct.js into your GitHub website files, in the same place as index.html.
2. Open index.html and add this line right before </body>:

<script src="buy-now-direct.js"></script>

3. Open buy-now-direct.js and replace:
YOUR_PAYPAL_CHECKOUT_LINK

with your real PayPal checkout/payment link.

What it does:
- Add to Cart still adds to cart.
- Buy Now does NOT add anything to cart.
- Buy Now does NOT open the cart.
- Buy Now goes straight to your checkout/payment link.
